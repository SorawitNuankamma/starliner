คำสั่งเงื่อนไข (Conditional Statement) ใช้ควบคุมการทำงานของโปรแกรมตามเงื่อนไขที่กำหนด

1. If และ Else

score = 70
if score >= 50:
    print("ผ่าน")
else:
    print("ไม่ผ่าน")

2. Elif

ใช้สำหรับตรวจหลายเงื่อนไข

n = 0
if n > 0:
    print("บวก")
elif n == 0:
    print("ศูนย์")
else:
    print("ลบ")

3. Match Case (Python 3.10 ขึ้นไป)

status = "ok"
match status:
    case "ok":
        print("ทำงานต่อ")
    case "error":
        print("หยุด")
