ลูป (Loop) ใช้สำหรับทำงานซ้ำ ๆ อย่างมีประสิทธิภาพ

For Loop

เหมาะสำหรับจำนวนรอบที่แน่นอน

for i in range(3):
    print(i)
สามารถใช้กับลิสต์หรือดิกชันนารี
```python
items = ["a","b"]
for item in items:
print(item)
```

While Loop

ทำงานจนกว่าเงื่อนไขจะเป็นเท็จ

count = 0
while count < 3:
    print(count)
    count += 1

แบบฝึกหัด

เขียนโปรแกรมรับคะแนนหลายตัวและคำนวณค่าเฉลี่ยโดยใช้ลูป
