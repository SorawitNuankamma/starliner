คลาสและออบเจกต์ (Class & Object) เป็นแนวคิดสำคัญของการเขียนโปรแกรมเชิงวัตถุ

การประกาศคลาส

class Person:
    def __init__(self,name):
        self.name = name
    def say(self):
        print(f"สวัสดี {self.name}")

การสร้างออบเจกต์และเรียกใช้เมธอด

p = Person("Bee")
p.say()
